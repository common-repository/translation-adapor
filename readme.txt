=== IOL translation ===
by iol8.com

== Requirement ==
- qTranslate 2.5.19+
- wordpress 3.8.1

== Installation ==
- To install it simply unzip the file linked above and save it in your plugins directory under wp-content. 
- In the plugin manager activate the plugin. 
- you will find this plugin on your wp admin left menu and inside post, page, category, tag's list and view detail page

== Uninstall ==
- uninstall from wp plugin menu
